# 👮 Command Center User Guide

The real-time dashboard gives you complete jurisdiction over traffic analytics and alerts generated. Start the system via scripts in Guide 02 beforehand!

## Default Administrator Logins

Navigate to **http://localhost:5173**. Utilize defaults on initial seed state:
**Email:** `admin@thirdeye.com`
**Pass:** `Admin@123`

## Navigating Verification

1. Watch the right-side notification tray for Socket-emitted real-time violation popups.
2. Click **Review** on the alert token.
3. Observe either a full high-fidelity Video Capture of Speedings/Wrong Directions, or a crisp AI tagged JPEG showing Helmetless details alongside paddle OCR plates.
4. If correct, smash the green `VERIFY & ISSUE CHALLAN`. This executes internal PDF generation methods on the Node.js backend to draft formal PDF challans ready for download.

## Advanced Actions

*   Rejecting records never deletes them permanently (crucial audit requirements tracking old status in memory). Supervisors log on to re-approve reversed decisions. 
*   Visit the Map Heatmap pane to dynamically map high-danger geographic zones globally loaded across all live nodes.
